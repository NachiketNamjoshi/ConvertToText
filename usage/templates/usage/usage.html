  • ConvertToText - Usage Guide

  • Once Installed ConvertToText, you can directly import ConvertToText.
  • A few example usage with all the file formats.

NOTE:

ConvertToText returns the string that has the converted text from any file format.

  • PDF

    Converting PDF can be done by using extract_text() in ConvertToText.pdfConvertor 
    from ConvertToText import pdfConvertor
print(pdfConvertor.extract_text(input("Absolute Path To File: ")))

    or you could just store the converted text into a variable and proceed from there on out.

    from ConvertToText import pdfConvertor
ConvertedText = pdfConvertor.extract_text(input("Absolute Path To File: "))

  • DOC:

    Converting a MS .doc file cant also be done by using extract_text() in ConvertToText.docConvertor

    from ConvertToText import docConvertor
print(docConvertor.extract_text(input("File Path: ")))

    or you could just store the converted text into a variable and proceed from there on out.

    from ConvertToText import docConvertor
ConvertedText = docConvertor.extract_text(input("File Path: "))
