  • ConvertToText - Installation Guide

To install ConvertToText follow the following steps:

  • Clone Source From Github (For Being Updated):

    git clone https://github.com/NachiketNamjoshi/ConvertToText

    OR Download it from RELEASES Page

  • Go To the root folder of the source and run setup.py as:

    python setup.py install

NOTE:

You MUST fulfill the minimum requirements to USE the Utility.
The following Libraries must be installed in order to use the respective file type conversion:

  • PDF -PyPDF - mstamy2/PyPDF2

CREDITS:
